# Smart Laundromat Concept
## Android App
* Makes HTTP requests to server
## Server
* Handles HTTP requests
* Manages machine statuses and queue
* To host, run from root directory server/.mvnw spring-boot::run
    * Make sure database credentials are set in smart-laundromat-concept\smart-laundromat-concept\server\target\classes\application.properties
